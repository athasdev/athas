# Documentation

Welcome to Athas, a lightweight code editor built with Tauri.

## What is Athas?

Athas is a lightweight, cross-platform code editor designed for developers who want speed without sacrificing functionality.

## Getting Started

- [Installation](installation) - Download and install Athas

## Technology Stack

Athas is built using Tauri. React on the frontend, Rust on the backend.

## Platform Support

Athas is cross-platform. Currently we have official builds for:

- macOS (Intel and Apple Silicon)
- Windows (64-bit)
- Linux (various distributions)
