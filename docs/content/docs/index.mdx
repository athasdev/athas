Welcome to Athas, a lightweight code editor built with Tauri.

## What is Athas?

Athas is a lightweight, cross-platform code editor designed for developers who want speed without sacrificing functionality.

## Why Athas?

- **Lightweight**: Fast startup and low resource usage
- **Cross-Platform**: Available on macOS, Linux, and Windows
- **Customizable**: Themes, keymaps, and extensions
- **Modern Tech**: Built with Tauri, React, and Rust
- **Open Source**: Community-driven development
- **AI Features**: Built-in AI assistant

## Getting Started

- [Installation](/docs/installation) - Download and install Athas

## Technology

Athas is built using Tauri. React on the frontend, Rust on the backend.

## Platform Support

- macOS (Apple Silicon & Intel)
- Linux
- Windows
