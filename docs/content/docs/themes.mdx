Athas supports custom themes to personalize your editing experience.

## Changing Themes

1. Open **Settings** (`Cmd+,` / `Ctrl+,`)
2. Go to **Appearance**
3. Select a theme from the dropdown

Or use the Command Palette:
1. Press `Cmd+Shift+P` / `Ctrl+Shift+P`
2. Search for **Change Theme**
3. Select a theme

## Built-in Themes

Athas includes several built-in themes for both light and dark modes.

## What Themes Affect

Themes define colors for:
- Editor background and foreground
- Syntax highlighting
- UI elements (sidebar, tabs, panels)
- Git indicators
- Terminal colors
