## Prerequisites

- Visual Studio Build Tools
- Rust (via rustup)
- Node.js 18+ and bun
- WebView2

## Install Dependencies

1. **Visual Studio Build Tools**
   - Download from [Visual Studio](https://visualstudio.microsoft.com/visual-cpp-build-tools/)
   - Install "Desktop development with C++"

2. **Rust**
   - Download rustup from [rustup.rs](https://rustup.rs/)
   - Run the installer

3. **Bun**
   ```powershell
   powershell -c "irm bun.sh/install.ps1 | iex"
   ```

4. **WebView2**
   - Usually pre-installed on Windows 10/11
   - If needed, download from [Microsoft](https://developer.microsoft.com/en-us/microsoft-edge/webview2/)

## Clone and Build

```powershell
git clone https://github.com/athasdev/athas.git
cd athas
bun install
bun dev
```
