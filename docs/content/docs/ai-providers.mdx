Athas supports multiple AI providers.

<Tabs items={["Cloud", "Local"]}>
  <Tab value="Cloud">
    ### OpenAI

    1. Get an API key from [OpenAI](https://platform.openai.com/api-keys)
    2. Go to **Settings > AI > API Keys**
    3. Click "Set API Key" next to OpenAI
    4. Paste your key and save

    ### Anthropic

    1. Get an API key from [Anthropic](https://console.anthropic.com/)
    2. Go to **Settings > AI > API Keys**
    3. Click "Set API Key" next to Anthropic
    4. Paste your key and save

    ### OpenRouter

    1. Get an API key from [OpenRouter](https://openrouter.ai/keys)
    2. Go to **Settings > AI > API Keys**
    3. Click "Set API Key" next to OpenRouter
    4. Paste your key and save

    ### Gemini

    1. Get an API key from [Google AI Studio](https://aistudio.google.com/app/apikey)
    2. Go to **Settings > AI > API Keys**
    3. Click "Set API Key" next to Gemini
    4. Paste your key and save

    ### xAI Grok

    1. Get an API key from [xAI Console](https://console.x.ai/)
    2. Go to **Settings > AI > API Keys**
    3. Click "Set API Key" next to xAI Grok
    4. Paste your key and save
  </Tab>
  <Tab value="Local">
    ### Ollama

    Run AI models locally using [Ollama](https://ollama.ai/):

    1. Install Ollama on your system
    2. Pull a model: `ollama pull codellama`
    3. Select "Ollama" as your provider in Athas settings
    4. Available models will be detected automatically

    <Callout type="info">
      Ollama must be running locally for model detection and requests to work.
    </Callout>

    Popular models:
    - **CodeLlama**: Optimized for code generation
    - **Llama 3**: General purpose
    - **Mistral**: Fast and capable
  </Tab>
</Tabs>

## Model Parameters

- **Temperature**: 0 = deterministic, 2 = creative
- **Max Tokens**: Limits response length
