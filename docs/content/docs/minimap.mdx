The minimap provides a condensed overview of your code on the right side of the editor.

## Enabling the Minimap

The minimap is **disabled by default**. Enable it via:

### Settings Panel

1. Open **Settings** (`Cmd+,` / `Ctrl+,`)
2. Go to the **Editor** tab
3. Toggle **Show Minimap** on

### Keyboard Shortcut

- **macOS:** `Cmd+Shift+M`
- **Windows/Linux:** `Ctrl+Shift+M`

### Command Palette

1. Open the Command Palette (`Cmd+Shift+P` / `Ctrl+Shift+P`)
2. Search for **Toggle Minimap**
3. Press `Enter`

## Minimap Interaction

- **Click** anywhere on the minimap to scroll to that position
- **Drag** the viewport indicator to scroll through the file
- The highlighted region shows your current viewport
