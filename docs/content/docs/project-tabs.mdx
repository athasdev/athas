---
title: Project Tabs
description: Manage multiple projects with tabs
---

Athas supports working with multiple projects simultaneously through project tabs.

## Opening Projects

- Click the **+** button in the title bar to open a new project
- Drag and drop a folder onto the editor window
- Each opened project appears as a tab in the title bar

## Switching Between Projects

- Click on any project tab to switch to that project
- Each project maintains its own set of open files

## Closing Projects

- Hover over a project tab to reveal the close button (X)
- Click the X to close that project

## Reordering Projects

- Click and drag any project tab to reorder it

## Right-Click Menu

- **Copy Path** - Copy the project's full path
- **Reveal in Finder** - Open in file manager
- **Close Project** - Close the selected project
- **Close Other Projects** - Close all except selected
- **Close to Right** - Close all to the right
