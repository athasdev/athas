title: Project Tabs
description: Manage multiple projects with tabs
---

Athas supports multiple projects in a single window using project tabs.

## Opening Projects

- Click the **+** button in the title bar to open a project
- Drag and drop a folder onto the editor window
- Each project appears as a tab in the title bar

## Switching Between Projects

- Click a tab to switch projects
- Each project keeps its own open files and state

## Closing Projects

- Hover a project tab to reveal the close button (X)
- Click the X to close the project

## Reordering Projects

- Click and drag any project tab to reorder it
