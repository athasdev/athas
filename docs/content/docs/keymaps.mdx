Athas allows you to customize keyboard shortcuts to match your workflow.

## Viewing Keymaps

1. Open **Settings** (`Cmd+,` / `Ctrl+,`)
2. Go to **Keyboard** tab
3. Browse available commands and their shortcuts

## Customizing Keymaps

Click on any shortcut to change it:
1. Press the new key combination
2. The shortcut updates immediately

## Default Shortcuts

### File Operations
- `Cmd/Ctrl+S` - Save file
- `Cmd/Ctrl+Shift+S` - Save all files
- `Cmd/Ctrl+W` - Close file
- `Cmd/Ctrl+N` - New file

### Navigation
- `Cmd/Ctrl+P` - Quick open file
- `Cmd/Ctrl+Shift+P` - Command palette
- `Cmd/Ctrl+G` - Go to line
- `Cmd/Ctrl+Shift+O` - Go to symbol

### Editing
- `Cmd/Ctrl+Z` - Undo
- `Cmd/Ctrl+Shift+Z` - Redo
- `Cmd/Ctrl+D` - Select word / next occurrence
- `Cmd/Ctrl+/` - Toggle comment
