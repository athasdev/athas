Athas includes a built-in AI assistant for coding help.

## Features

- **Chat Interface**: Conversational AI for coding questions
- **Code Completion**: AI-powered code suggestions
- **Multiple Providers**: OpenAI, Anthropic, Ollama, and more
- **Context Awareness**: AI understands your project files

## Getting Started

1. Open the AI panel from the sidebar (or `Cmd+Shift+A` / `Ctrl+Shift+A`)
2. Configure your AI provider in **Settings > AI**
3. Start chatting or asking questions about your code

## Supported Providers

- **OpenAI**: GPT-4, GPT-4 Turbo, GPT-3.5 Turbo
- **Anthropic**: Claude 3.5 Sonnet, Claude 3 Opus, Claude 3 Haiku
- **Ollama**: Local models (Llama, CodeLlama, Mistral, etc.)

## Configuration

Configure AI settings in **Settings > AI**:

- **Provider**: Select your preferred AI service
- **Model**: Choose the specific model
- **Temperature**: Control response creativity (0-2)
- **Max Tokens**: Limit response length
