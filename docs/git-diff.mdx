---
title: Diff Viewer
description: View and compare file changes
---

Athas includes a powerful diff viewer.

## Viewing Diffs

### From Source Control

1. Open the Source Control panel
2. Click on any changed file
3. The diff viewer opens showing changes

### From Commit History

1. Open the commit history
2. Click on any commit
3. Click on a file to view its diff

## Diff Modes

### Inline Diff

Shows additions and deletions inline:
- Green background for additions
- Red background for deletions

### Side-by-Side Diff

Shows old and new versions side by side:
- Left pane: original
- Right pane: modified

## Gutter Indicators

In the editor, the gutter shows Git change indicators:

- **Green bar**: Added lines
- **Red bar**: Deleted lines
- **Yellow bar**: Modified lines

Click on these indicators to see the original content.

## Inline Blame

Enable inline blame in **Settings > Editor** or toggle with `Cmd/Ctrl+Shift+B`.
