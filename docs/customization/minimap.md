# Minimap

The minimap provides a condensed overview of your code on the right side of the editor, making it easy to navigate large files at a glance.

## Enabling the Minimap

The minimap is **disabled by default**. You can enable it in several ways:

### Settings Panel

1. Open **Settings** (`Cmd+,` / `Ctrl+,`)
2. Go to the **Editor** tab
3. In the **Display** section, toggle **Show Minimap** on

### Keyboard Shortcut

- **macOS:** `Cmd+Shift+M`
- **Windows/Linux:** `Ctrl+Shift+M`

### Command Palette

1. Open the Command Palette (`Cmd+Shift+P` / `Ctrl+Shift+P`)
2. Search for **Toggle Minimap**
3. Press `Enter` to toggle

## Minimap Interaction

- **Click** anywhere on the minimap to scroll the editor to that position
- **Drag** the viewport indicator to scroll through the file
- The highlighted region shows your current viewport in the editor

## Theme Support

The minimap automatically uses your current theme's syntax colors. When you switch themes, the minimap colors update to match.
