Athas is a lightweight, cross-platform code editor focused on speed and modern workflows.

## Quick Start

<Steps>
  <Step>
    [Install Athas](/docs/installation)
  </Step>
  <Step>
    Open a project via the **+** button in the title bar or drag a folder into the window
  </Step>
  <Step>
    Use the Command Palette (`Cmd/Ctrl+Shift+P`) to discover actions
  </Step>
</Steps>

## Why Athas?

- **Fast**: Quick startup and low resource usage
- **Cross-platform**: macOS, Linux, and Windows
- **Customizable**: Themes, keymaps, and extensions
- **Modern**: Built with Tauri, React, and Rust
- **Open-source**: Community-driven development
- **AI-ready**: Optional AI assistant with multiple providers
