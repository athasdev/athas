Athas includes a built-in AI assistant for coding help.

## Features

- **Chat Interface**: Conversational AI for coding questions
- **Multiple Providers**: OpenAI, Anthropic, OpenRouter, Gemini, Grok, Ollama, and more
- **Context Awareness**: AI can use files you reference

## Getting Started

<Steps>
  <Step>
    Open the AI panel from the sidebar (or `Cmd+Shift+A` / `Ctrl+Shift+A`)
  </Step>
  <Step>
    Configure your AI provider in **Settings > AI**
  </Step>
  <Step>
    Start chatting or asking questions about your code
  </Step>
</Steps>

## Configuration

Configure AI settings in **Settings > AI**:

- **Provider**: Select your preferred AI service
- **Model**: Choose the specific model
- **Temperature**: Control response creativity (0-2)
- **Max Tokens**: Limit response length
