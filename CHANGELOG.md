# Changelog

All notable changes to this project are documented in this file.

## [0.4.1] - 2026-03-02

### Added
- Better ACP agent binary detection for Go and Cargo runtimes.
- Manual Git repository selection support.

### Changed
- Refined workspace, Git, terminal, editor, and AI chat workflows.
- Improved ACP robustness and streaming behavior.
- Flattened and reorganized documentation structure.

### Fixed
- Parser query fallback and predicate compatibility handling.
- Inline edit flow and AI message rendering issues.
- File explorer handling for `.DS_Store`.
- Clippy/lint pre-release issues in fuzzy matching, ACP bridge, and tool installer paths.

### Internal
- Updated CI and release-related scripts/workflows.
